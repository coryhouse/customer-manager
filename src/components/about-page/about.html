About

Assumptions

  1. Connectivity won't be an issue
  2. IE8+ is supported
  3. Team is confortable with JS (otherwise, would've went with MVC & Razor)

Technologies

  • Knockout - Two way binding and components
  • RequireJS - Module system
  • GulpJS - Build system
  • Bower - Package management
  • Node and NPM - To install dependencies
  • Yeoman - Scaffolding
  • Jasmine - JS Unit testing library
  • Karma - Unit test runner
  • Bootstrap - HTML, CSS, JS framework

Build Process

  1. Open command line
  2. Change directory to the root of this project
  3. Type npm install to install all necessary packages (stored in package.json)
  4. Type "gulp". This will run the default task in gulp which does the following:
    • Minifies and concatenates all JS
    • Migrates built code to the dist folder
  5. Fire up your favorite webserver and point it at /dist

Tests

You can run tests in one of two ways:

  1. Open /test/index.html in your browser.
  2. Type "install -g karma-cli" so that you can run Karma from anywhere. Then type "karma start" in the project root dir (where karma.conf is stored).
